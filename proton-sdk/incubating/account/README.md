# Account implementation for the Drive SDK

This module provides account login, session management, and account API access. It can be used in combination with the Proton Drive SDK to provide account functionality to the client. However, it might not have all the features required.

This module is not planned to be published or promoted outside of the incubating directory. It lives here only for convenience of the Proton Drive SDK development to provide a temporary solution until the Account SDK is available.
